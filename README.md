Установка
===============================================================================
Для запуска игры нужны библиотеки: pygame, pytmx, pyganim. Для их установки в терминале введите команды:

    pip pip install pygame
    
    pip install pyganim
    
    pip install pytmx
    
Дизайн уровней:
===============================================================================
Карты уровней создаются в программе Tiled (www.mapeditor.org)
